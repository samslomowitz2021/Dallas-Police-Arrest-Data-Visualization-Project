Dallas Police Arrests

Use the interactive charts below to explore the dataset

Inspiration

We want to build a interactive dashboard using JavaScript libraries to visualize police arrest data in Dallas, TX. Our main filters will be racial category of arrestees. We would like to design an interactive map similar to the UK Police Tableau Public dashboard showing the location of the incidents colored by type of crime.

UK Dataset Example


Our main questions are:

  • 1. Do males proportionality outnumber females in arrest data?
  • 2. Which race categories are more frequent in arrests?
  • 3. Is Dallas a safe/low-crime city?
  • 4. Which days of the week or years had the lowest/highest arrest count?
  • 5. Which areas of Dallas are high/low in arrest count?

Objective

To create an interactive dashboard to showcase real-world law-enforcement data and visualize crime, location, and arrest data in the DFW metroplex.
