Dallas Police Arrests

Use the interactive charts below to explore the dataset

Inspiration

We want to build a interactive dashboard using JavaScript libraries to visualize police arrest data in Dallas, TX. Our main filters will be year (2014 to 2022) and zip code. We would like to design an interactive map similar to the UK Police Tableau Public dashboard showing the location of the incidents colored by type of crime.

UK Dataset Example


Our main questions are:

  • 1. Do drug-related arrests outnumber violent crimes? We would like to look at any overlap between these categories.
  • 2. Do males proportionality outnumber females in arrest data?
  • 3. Which race categories are more frequent in arrests?
  • 4. What drug categories are abundant in the data?
  • 5. Is Dallas a safe/low-crime city?
  • 6. Which areas of Dallas are high in arrest count?
  • 7. What time of day and what day of the week do most arrest occur?

Objective

To create an interactive dashboard to showcase real-world law-enforcement data and visualize crime, location, and arrest data in the DFW metroplex.
